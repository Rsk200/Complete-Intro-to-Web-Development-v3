# Yaad's Personal Blog

## Description
This project is a complete, professional personal blog website built using semantic HTML5. It serves as a platform to document my journey as a Computer Science student passionate about Artificial Intelligence, Machine Learning, and software development. The blog features articles about my learning experiences, projects, and future career goals in AI research.

This project was built as part of my learning journey through the **[Complete Intro to Web Development v3](https://frontendmasters.com/courses/web-development-v3/)** course from **Frontend Masters**.

## Features
* Multi-page website
* Five blog posts
* About page
* Semantic HTML5
* Accessibility support
* Images
* Responsive-ready structure

## How To Run
To view the blog, simply download or clone the repository to your local machine. Open the `index.html` file in any modern web browser (such as Google Chrome, Mozilla Firefox, or Microsoft Edge). 
Alternatively, you can drag and drop the `index.html` file directly into your browser window.
