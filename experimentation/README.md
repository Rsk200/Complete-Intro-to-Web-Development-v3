# Character Repeater Exercise

Welcome to the character repeater exercise! 🚀

This is a simple exercise to demonstrate how to use variables and loops in JavaScript. 
The code takes a character and repeats it a specified number of times using a `for` loop.

## What's inside?

In the `experimentation.js` file, you will find a beginner-friendly code snippet that does the following:

1.  **Declares variables:** We set up a string variable to keep track of our results.
2.  **Uses a loop:** A `for` loop is used to repeat an action. In this case, adding a character to our string.
3.  **Logs to the console:** After the loop finishes, we use `console.log()` to see the final result.

## Examples we tried:
- Repeating `'f'` 5 times outputs: `fffff`
- Repeating `'a'` 10 times outputs: `aaaaaaaaaa`
- Repeating `'c'` 100 times outputs: `cccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccccc`
- Repeating `'🐶'` 3 times outputs: `🐶🐶🐶`

## How to run the code
You can run this code by executing the following command in your terminal:
```bash
node experimentation.js
```
